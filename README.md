# 📊 Kumpulan Tugas Visualisasi Data dan Informasi

Repository ini berisi kumpulan tugas dan project praktikum terkait visualisasi data dan informasi menggunakan berbagai teknik analisis data serta tools visualisasi. Project dibuat sebagai bagian dari proses pembelajaran di bidang Data Science dan Data Visualization.

## 🚀 Tentang Repository

Repository ini mendokumentasikan berbagai eksplorasi visualisasi data untuk mengubah data mentah menjadi insight yang lebih mudah dipahami secara visual. Fokus utama project meliputi:

- Data cleaning dan preprocessing
- Exploratory Data Analysis (EDA)
- Visualisasi data interaktif dan informatif
- Interpretasi pola dan insight dari data
- Penyajian data untuk kebutuhan analisis dan pengambilan keputusan

Melalui repository ini, saya mengembangkan kemampuan dalam:
- Data Analysis
- Data Visualization
- Problem Solving berbasis data
- Storytelling menggunakan visualisasi
- Pengolahan dataset menggunakan Python dan tools analitik lainnya
